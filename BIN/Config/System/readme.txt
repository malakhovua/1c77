Системные файлы для скриптов

dynwrap.dll         - COM-dll для вызовов функций WinAPI из скриптов
macrosenum.dll      - объект-итератор для перебора скриптов ОпенКонфа
SelectValue.dll     - реализация различных вариантов диалога для выбора значания
svssvc.dll          - сервисные функции для использования в скриптах
WshExtra.dll        - дополнительные функции для использования в скриптах (Clibboard etc.)
CommonServices.wsc  - скриптлет - библиотека общих функций, используемых в скриптах для OpenConf
scriptlet.tlb       - библиотека типов для CommonServices.tlb
