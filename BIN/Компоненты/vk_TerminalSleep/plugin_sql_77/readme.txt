Установка

Скопировать файлы:
odbc33.dll 
odbc33.ini
patch_bkend.exe

в папку 1Cv77\BIN 

Запустить patch_bkend.exe

Отмена установки - переустановить 1С, или вернуть пропатченную bkend.dll из BkEnd.dll.bak

Тестировалось на 25 релизе 1С:Предприятие (на более поздних релизах, если там не последует изменений по части этой DLL, тоже должно работать).
В Windows 98 если возникнет ошибка, необходимо установить драйвер MDAC посвежее, скачав его с сайта Microsoft.
