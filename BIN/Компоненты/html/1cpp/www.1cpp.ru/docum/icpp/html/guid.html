Класс предназначен для работы с глобальным уникальным идентификатором (GUID). После создания объект содержит значение пустого идентификатора.

Методы

ИзСтроки / FromString

Синтаксис: ИзСтроки(ИдентификаторСтрокой)

Параметры:

  • ИдентификаторСтрокой - тип: Строка. Строка для загрузки, должна быть в каноническом виде для представления идентификатора.

Описание: позволяет загрузить объект из строки.

ВСтроку / ToString

Синтаксис: ВСтроку()

Возвращает: тип: Строка. Cтроковое представление идентификатора в каноническом виде.

Описание: конвертирует содержимое объекта в строку.

Пустой / IsNil

Синтаксис: Пустой()

Возвращает: тип: Число. 1 - пустой, 0 - не пустой.

Описание: позволяет проверить содержимое объекта - пустой или нет.

УстПустой / SetNil

Синтаксис: УстПустой()

Описание: формирует пустой идентификатор в объекте.

Равны / IsEqual

Синтаксис: Равны(Идентификатор)

Параметры:

  • Идентификатор - тип: Строка, GUID. Строка или объект для сравнения.

Возвращает: тип: Число. 1 - равны, 0 - не равны.

Описание: позволяет сравнить два идентификатора.

Новый / New

Синтаксис: Новый()

Описание: создает новый идентификатор в объекте.
